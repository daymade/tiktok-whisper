## 💡 使用建议

### 🎯 第一次使用
1. 阅读 **[快速开始](快速开始.md)**
2. 运行 `uv run python export_to_md.py config --show` 检查配置
3. 试试导出一个小数据集
4. 遇到问题查看 **[故障排除指南](故障排除指南.md)**

### 🔄 日常使用
记住这些常用命令：
```bash
# 查看用户
uv run python export_to_md.py list-users

# 导出数据
uv run python export_to_md.py export --user "用户名"
```

### 🚀 高级使用
- 学习配置文件的各种选项
- 创建自己的快捷脚本
- 批量导出多个用户
